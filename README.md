# Project Hippocrates

> AI-powered Personal Learning Operating System for TUS.

## Vision

Project Hippocrates aims to become an AI-powered Second Brain that helps medical students learn more efficiently by integrating books, videos, question banks, exams, flashcards, knowledge graphs, and personalized learning analytics into a single intelligent ecosystem.

## Status

🚧 Phase 0 — Foundation

This repository is currently under active architecture and system design.

Code development has **not** started yet.

The first priority is building a robust technical architecture and Master Specification.
