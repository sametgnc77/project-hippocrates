# Database Overview

## Purpose

The database stores the intelligence of MENTIS.

It does NOT store raw learning materials.

Raw materials remain inside the Source Library.

The database stores processed knowledge.

---

# Database Layers

## Layer 1

User Data

Examples

- User
- Settings
- Progress

---

## Layer 2

Learning Data

Examples

- Concepts
- Topics
- Branches
- PACER
- Flashcards

---

## Layer 3

Knowledge Graph

Stores relationships between concepts.

---

## Layer 4

Learning Analytics

Stores

- Study sessions
- Reviews
- Wrong answers
- Statistics

---

## Layer 5

AI Memory

Stores

- Embeddings
- AI summaries
- AI confidence
- AI generated metadata

---

# Design Rule

Raw files are never stored inside the database.

Only structured learning intelligence is stored.
